# 🚀 PathWise AI

## AI-Powered Career Decision Intelligence Platform

PathWise AI is an AI-powered career assistant built using **Python, Streamlit, and Google Gemini AI**. It helps students and job seekers analyze their resumes, identify skill gaps, and receive personalized career guidance.

---

## ✨ Features

- 📄 Upload Resume (PDF)
- 🤖 AI Resume Analysis
- 📊 Career Readiness Score
- 💪 Resume Strengths & Weaknesses
- 🎯 Missing Skills Detection
- 📚 Personalized Learning Roadmap
- 🏆 Certification Recommendations
- 💬 AI Career Chat Assistant

---

## 🛠️ Tech Stack

- Python
- Streamlit
- Google Gemini API
- PyPDF

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/sami-tech840/pathwise-ai.git
```

Go into the project:

```bash
cd pathwise-ai
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Create a `.env` file and add your Gemini API key:

```env
GEMINI_API_KEY=YOUR_API_KEY
```

Run the application:

```bash
streamlit run app.py
```
